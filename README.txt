READ ME

This app goal is to perform CRUD operation read the following 

Site1.Master: is The main page to the web form it's goal is to 
bind multiple pages to a master page to avoid code reuse

Default.aspx: here is where all the logic is located
here is where you see and interact with the main form

DAO: is the connection between the db and the program

Dataconfig.cs: it specifies the connectionstring.

IMPORTANT: 
when running please change the connection string uid and pwd 
under Web.config to your own in order to run the app.    

also please run the crud_user.sql file and crud_routines
to create the database and its procedures
